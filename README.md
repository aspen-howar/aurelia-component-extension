# Unified Aurelia Components Extension

## Solutions

* Everytime I want to create a new Aurelia component, I have to manually create multiple files and set up boilerplate code.

  **Solution:** This extension provides some context menu options to automate the creation of a new Aurelia component. It generates the necessary HTML, TypeScript, and less files with boilerplate code, saving you time and effort.

* We have many basic UI components that I ofter forget their configurations and usages.

  **Solution:** The extension includes snippets for commonly used Aurelia components. You can easily insert these snippets into your code, helping you remember the configurations and usages of basic UI components.

* We have many custom components in our Aurelia projects, if want to find their real location, have to search.

  **Solution:** The extension includes a definition provider that allows you to quickly navigate to the definition of custom Aurelia components directly from your HTML files.
